Criar Trigger quando um meeting é iniciado
-> Ele adiciona os itens da sala aos itens do encontro
-> Adiciona os itens do exercicio aos itens do encontro
-> Desconta a quantidade do item do inventory

Criar Trigger quando um exercicio é adicionado ao encontro já iniciado
-> Adiciona os itens do exercicio ao encontro
-> Desconta a quantidade do item do inventory

Criar Trigger quando adicionar um item em uma reunião

Criar Trigger quando um meeting é finalizado
-> Adiciona a quantidade novamente dos itens na sala
-> Adiciona a quantidade novamente dos itens do exercicio se o item não tem o sigle use

Criar Trigger quando a sala for criada para retirar a quantidade do estoque
